HMT Case Codes and their meaning
HMT Case Codes
HMT watches have generally 4 or 6 digitals case codes embossed on back side of case. Each case code is unique in a way and there is no prescribed formula to decipher the case code. They generally cnsists of 3 things : Factory Code - 2 digits generally Case Type - 2 digits generally Year of manufacturing - not present in 4 digit case codes
HMT watch Factory Codes
01 and 02 - Bangalore factories 03 - Chinar factory in Jammu and Kashmir 04 - Tumkur factory in Karnataka 05 - Ranibagh factory in Uttaranchal
4 Digit Case Codes
Chinar watches that you see available on the site these days typically have a four digit case code and the case type code is not included.
